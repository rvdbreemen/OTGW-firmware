De volgende stappen om de bestanden te uploaden naar LittleFS:

1. Installeer de LittleFS-upload plugin voor de Arduino IDE

Link: https://github.com/earlephilhower/arduino-esp8266littlefs-plugin

2. Start de Arduino IDE opnieuw op, en ga naar het het Hulpmiddelen menu, klik daar op de "ESP8266 LittleFS Data Upload" menuitem om de **data** directory naar de LittleFS van de NodeMCU te uploaden.

3. Als u deze stappen heeft uitgevoerd, start u uw ESP8266 opnieuw op. Sluit hem daarna weer aan op je OTGW en sluit hem aan.

Indien je reeds een werkende firmware op de ESP8266 hebt, kan je ook Over-The-Air (OTA) updates van de LittleFS data uitvoeren. In het file explorer menu, druk op de knop [Update Firmware], daar kan je naast de firmware binary, ook de littlefs binary uploaden.
