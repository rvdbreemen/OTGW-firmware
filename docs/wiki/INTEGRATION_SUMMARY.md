## GitHub Actions Integration

### Sync Workflow
File: `.github/workflows/sync-wiki.yml`

**Trigger**: Push to `docs/wiki/` directory

**Process**:
1. Clone GitHub wiki repository
2. Copy all .md files from `docs/wiki/` to wiki
3. Update `_Sidebar.md` with v3 section
4. Commit and push to wiki repository
5. Report sync status

**Benefits**:
- ✅ Single source of truth in repository
- ✅ Version control for all wiki content
- ✅ Pull request workflow for changes
- ✅ Automatic sync on merge
- ✅ Audit trail of all changes
