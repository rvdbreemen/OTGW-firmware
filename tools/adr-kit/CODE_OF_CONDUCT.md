# Code of conduct

This project adopts the **[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full canonical text is maintained at the link above. We do not vendor a copy here; the upstream version is the authoritative one and is updated by its maintainers when needed.

## In short

- Be respectful and constructive in issues, pull requests, and discussions.
- Give credit where it is due (the upstream-attribution chain in this repo is a worked example).
- Disagree on the technical merits, not on the person.

## Reporting

If you experience or witness behaviour that violates the Contributor Covenant in any space associated with this project, please email the maintainer at <robert@vandenbreemen.net> with `adr-kit conduct:` in the subject line.

You will get an acknowledgement within 7 days. Reports are handled privately and the reporter's identity is kept confidential to the extent compatible with addressing the issue.

## Scope

This code of conduct applies to all project spaces (issues, pull requests, discussions, code review, documentation comments) and to one-on-one communication where the project is the context.

## Enforcement

For enforcement guidelines, see the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines). The maintainer reserves the right to apply the standard ladder of consequences (correction, warning, temporary ban, permanent ban) at their discretion based on the severity and context of the violation.
