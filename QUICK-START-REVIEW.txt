================================================================================
  CRITICAL FINDINGS (All are FIXES, not new bugs)
================================================================================

1. ⭐⭐⭐ SERIAL BUFFER OVERFLOW PROTECTION
   - Previous: Silent data loss and corruption
   - Now: Detected, logged, recovered gracefully
   - Impact: Prevents data loss under high load

2. ⭐⭐⭐ GPIO OUTPUT BUG FIX
   - Previous: Never worked (used XOR instead of bit shift)
   - Now: Works correctly (1 << bit)
   - Impact: GPIO outputs now functional

3. ⭐⭐⭐ MEMORY SAFETY OVERHAUL
   - Previous: String class causing heap fragmentation
   - Now: Stack-allocated char[] buffers
   - Impact: More stable over long uptimes

================================================================================
  NEW FEATURES
================================================================================

- NTP time sync control (opt-in via NTPsendtime setting)
- Manual-only PIC firmware updates (no auto-checks)
- Wi-Fi configuration improvements (better UX)
- Password masking in Web UI
- REST API hardening (input validation)
- XSS protection (HTML/URL escaping)
