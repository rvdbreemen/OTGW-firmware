## Description

Volledige audit van de port van Python SAT (thermo-nova branch) naar C++ firmware. Dekt feature completeness, algoritme equivalentie, geheugen veiligheid, OTGW32 compatibiliteit, ADR review en API documentatie. Elke audit taak produceert follow-up fix-taken (label: audit-fix) voor gevonden afwijkingen.
