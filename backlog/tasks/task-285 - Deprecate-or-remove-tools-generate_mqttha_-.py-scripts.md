## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
ADR-077 (Accepted) declares the Python generators tools/generate_mqttha_data.py, tools/generate_mqttha_progmem.py and tools/generate_mqttha_readable.py as dead code relative to the shipped firmware. They reference the archived data/mqttha.cfg (now at docs/archive/mqttha.cfg) and produce outputs that are no longer used by the build. Keeping them in the repository without a notice creates two risks: new contributors waste time trying to use them, and bitrot leaves them silently broken. Resolve by either deleting them, or adding an explicit deprecation header referencing ADR-077.
<!-- SECTION:DESCRIPTION:END -->
